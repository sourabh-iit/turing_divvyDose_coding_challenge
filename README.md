## Install:

You can use a virtual environment:
```
pip install -r requirements.txt
```

## Running the code



### Spin up the service

```
# activate the environment and start up local server
python -m run 
```

### Making Requests

```
curl -i "http://127.0.0.1:5000/merge-profiles/<team/organization>"
```
